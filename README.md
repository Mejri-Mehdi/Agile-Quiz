<h1 align="center">🎯 Agile & Scrum Multiplayer Quiz 🚀</h1>

<p align="center">
  <strong>A real‑time multiplayer quiz game that tests and sharpens your Agile & Scrum knowledge.<br>Built with modern web technologies and powered by Firebase.</strong>
</p>

---

## 🌟 Features

- ⚡ **Real‑time Multiplayer** – join a lobby, see who else is playing, and compete simultaneously
- 🎨 **Stunning UI** – custom glassmorphism design, fluid animations, and a fully responsive layout
- 📊 **Dynamic Scoring** – live score updates, visual progress bars, and a final leaderboard
- 🎮 **Host Controls** – the first player becomes the host and manages the game flow
- 📱 **Mobile Friendly** – looks and works great on phones, tablets, and desktops

---

## 🛠️ Technologies Used

| Layer         | Technology / Library |
|---------------|----------------------|
| **Frontend**  | HTML5, CSS3 (custom glassmorphism), Vanilla JavaScript (ES6+) |
| **Backend (BaaS)** | Firebase Realtime Database (game state & player sync) |
| **Hosting**   | Static host ready – GitHub Pages, Vercel, Netlify |

No complex frameworks or build tools – just pure, fast, and elegant code.

---

## 🚀 How to Run

1. **Clone the repository**
   ```bash
   git clone https://github.com/Mejri-Mehdi/AgileScrumQuiz.git
2. Open index.html in your favorite browser.

3. Enter your name in the username‑input field to join the lobby.

4. Share the game URL (or a QR code) with friends so they can join.

5. The host (first player to join) clicks “Lancer la partie” to start the quiz!

🔧 **Firebase configuration** – the project already includes the required Firebase config.
If you want to use your own Firebase project, replace the credentials in the JavaScript file.

⚡ Or skip the setup and play instantly:
🔗 https://mejri-mehdi.github.io/Agile-Quiz/
