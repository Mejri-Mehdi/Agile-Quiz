# Agile & Scrum Multiplayer Quiz 🚀

A real-time multiplayer quiz game designed to test and improve knowledge of Agile and Scrum methodologies. Built with modern web technologies and Firebase.

## 🌟 Features

- **Real-time Multiplayer**: Join a lobby, see other players, and compete in real-time.
- **Interactive UI**: Glassmorphism design, smooth animations, and responsive layout.
- **Dynamic Scoring**: Live score updates, visual progress tracking, and a final leaderboard.
- **Host Controls**: The first player to join becomes the host and controls the game flow.
- **Mobile Friendly**: Fully responsive design that works on all devices.

## 🛠️ Technologies Used

- **Frontend**: HTML5, CSS3 (Custom Glassmorphism), JavaScript (ES6+)
- **Backend (BaaS)**: Firebase Realtime Database (for game state and player synchronization)
- **Deployment**: Ready for static hosting (GitHub Pages, Vercel, Netlify)

## 🚀 How to Run

1. Clone the repository.
2. Open `index.html` in your browser.
3. Enter your name using the `username-input`.
4. Share the URL or QR code with friends to let them join.
5. The host (first player) clicks "Lancer la partie" to start!
